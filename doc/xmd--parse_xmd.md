# `parse_xmd`
**Synopsis**

```cpp

```

Parses xmd source recursively and collects all subentities and text.


## Parameter
**`xmd_lines`** &#8213; A list of strings containing the lines of the xmd source  
## Return Value

An `Entity` that contains the parsed contents
