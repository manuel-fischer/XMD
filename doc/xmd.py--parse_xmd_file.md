# `parse_xmd_file`
<small>*Function* &nbsp; - &nbsp; ["xmd.py"](../xmd.py)</small>

```python

```

Parse the `.xmd` file named by `ifile`.

## Parameters
**`cwd`** &#8213; The directory in which there is a file named `ifile`.
