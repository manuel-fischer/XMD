# `delete_file`
<small>*Function* - **file management**</small>  
**Synopsis**

```cpp
delete_file(fname):
```

Delete a (text) file.
The filename is echoed to `stdout`.

## Parameter
**`fname`** &#8213; The path of the text file.  
## Return Value

None.

## Side Effects

Deletes the file.
